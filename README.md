# util-file-filter

Утилита для фитрации содержимого файлов

Сборка:
- java 17+
- maven 3.8+

Зависимости:
- lombok 1.18.34 - https://projectlombok.org/

Сборка jar-файла:
- mvn clean package

Запуск утилиты:
- java -jar target/util-file-filter-1.0-SNAPSHOT-jar-with-dependencies.jar [OPTIONS] file1.txt file2.txt ...
